If Twitter is “micro-blogging”, then a tweet about a restaurant is a...

Micro-Review!

This is a project based on harnessing the power of these spur-of-the-moment reviews.

What do these reviews give us?

  • Popularity
  • Busy Hours
  • Sentiment
  • Pictures!

Our Approach

We're harvesting Twitter and Instagram data

We're assigning each review a rating...

...and each rating a restaurant...

...and we're mapping it

Next Steps with our Data

Tweet geolocation is incredibly inaccurate, so we need to implement some methods to use contextual clues to help geolocate.

We need to scrub the data for non-restaurant related tweets (such as political tweets).
